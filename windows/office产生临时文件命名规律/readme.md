# office产生临时文件命名规律

```
会产生~$文件的有doc,docx,rtf,docm,pptx,xlsx,xlsm,ppsx,potm,potx,ppsm,xml等
不会产生~$文件的有ppt,xls,csv等

doc/docx/rtf等文件产生的~$文件命名规则(pptx,xlsx虽然也会产生~$文件，但命名直接在前面加~$就好了)
计算文件名长度(如果有后缀名，需要忽略：doc忽略掉".doc"，docx忽略掉".docx"等)
如果文件名长度小于等于6，则直接在前面加"~$"
如果文件名长度大于6且小于等于8，把文件名从左到右删减到剩下6个字符，前面加"~$"
如果文件名大于8，删除前2位，在前面加"~$"
```
