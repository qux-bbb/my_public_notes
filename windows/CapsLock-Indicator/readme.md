# CapsLock Indicator

keywords: 键盘状态  

CapsLock Indicator, 可以提示 Num lock, Caps lock, Scroll lock 的状态，绿色表示启用。  
最好放在一个单独的文件夹下运行，因为下载新图标包时会保存在exe同一目录下。
