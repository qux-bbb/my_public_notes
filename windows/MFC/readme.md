# MFC

MFC, Microsoft Foundation Class  

Microsoft基础类库（MFC）是用于在Microsoft Windows中编程的"应用程序框架"。MFC提供了很多代码，这些代码是以下代码所必需的：  
- 管理窗口
- 菜单和对话框
- 执行基本输入/输出
- 存储数据对象的集合等

通过将应用程序特定的代码添加到MFC框架中，可以很容易地扩展或覆盖C++应用程序中的MFC框架的基本功能。
