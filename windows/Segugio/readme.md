# Segugio

github地址: https://github.com/reecdeep/segugio  

Segugio, 意大利语，猎犬的意思。  
使用yara扫描内存，匹配到之后，保存内存为文件，提取恶意软件配置。  

需要安装python，从github下载ConfigExtractors、YaraRules，修改settings.ini再使用。  
配置后待处理程序之后，点一下中间的狗头就可以运行了，运行起来比较卡，没有停止的按钮，把窗口关掉才会停止。
