# windows-terminal

## 前言
windows terminal 是微软开发的新终端  
官方文档：https://docs.microsoft.com/zh-cn/windows/terminal/  
Win + R，输入 "wt"，可快速打开windows terminal  


## 快捷键

切换 tab：`ctrl + tab`  
新的 tab：`ctrl + shift + t`  
切分窗格：`alt + shift + d`  
垂直方向切分窗格：`alt + shift + +`  
水平方向切分窗格：`alt + shift + -`  
关闭窗格：`ctrl + shift + w`  

搜索：`ctrl + shift + f`  


## 配置文件设置
"useAcrylic" 设置为true，可调节背景透明度，颜值爆表  
"acrylicOpacity" 调节具体的透明度
