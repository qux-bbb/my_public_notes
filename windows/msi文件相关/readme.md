# msi文件相关

msi, microsoft installer，windows使用的一种安装包文件  


## 创建msi安装包

大概流程：  
1. 在Visual Studio 扩展商店搜索安装 `Microsoft Visual Studio Installer Projects` 扩展(关键字搜 `Setup` 就好了)
2. 创建 `Setup Project`
3. 添加要安装的文件或文件夹，要设置的快捷方式

然后生成解决方案就好了
