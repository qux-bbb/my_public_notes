# netmonitor  

Network Monitor 3.4 is the archive versioned tool for network traffic capture and protocol analysis.  

下载地址:  
https://www.microsoft.com/en-us/download/details.aspx?id=4865  

安装包名称:  
```r
NM34_x86.exe
NM34_ia64.exe
NM34_x64.exe
```

特点是可以看到流量是具体哪个程序产生的  

不过只能看http流量，https流量虽然也显示，但还是加密的
