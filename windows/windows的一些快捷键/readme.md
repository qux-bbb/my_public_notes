# windows的一些快捷键

```r
Ctrl + A    全选
Ctrl + C    复制
Ctrl + V    粘贴
Ctrl + X    剪切
Ctrl + Z    撤销
Ctrl + F    查找
Ctrl + S    保存

Alt + Tab   切换程序
Alt + F4    关闭程序，关机

Win + L     锁屏
Win + E     打开文件管理器
Win + R     运行（打开cmd，regedit什么的）
Win + D     所有应用最小化，回到桌面（再按回到之前状态）
Win + V     打开剪贴板
Win + A     打开操作中心
Win + I     打开设置
Win + Shift + S	截屏
```
