# ctf线下赛相关

基本上每个队的环境都是一样的，所以相当于白盒审计  

攻击就是找漏洞  

防御分3部分：  
1. 改初始密码  
  netstat 或者 ps -aux 看进程，根据开放服务改相应密码  
  如 ssh mysql http  
2. 将可写目录降权  
3. 修改为正确配置
