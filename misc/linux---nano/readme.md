# linux---nano

nano是一个简单的编辑器，打开后可以在下方看到操作快捷键。  

快捷键用如下方式进行表示：  
控制键序列使用一个“^”符号标记，它可以用 Ctrl 键或按 Esc 键两次的方式进行输入。  
Meta 键序列使用“M-”符号标记，它可以用 Alt、Cmd 或 Esc 键输入，具体取决于键盘设置。  
另外，按 Esc 两次之后再键入从 000 到 255 之间的三位数字，则会输入该 ASCII 码对应的字符。  

`^G` 意思就是 `Ctrl + G`  
`M-6` 可以是 `Alt + 6`  

配置文件在 `/etc/nanorc`
