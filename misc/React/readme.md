# React

React 是一个前端框架。  
官网: https://react.dev/  


## 简单信息
创建应用命令：  
```
npx create-react-app hello
```

进入项目文件夹，启动应用：  
```
npm start
```

全局安装json服务器：  
```
npm install -g json-server
```

应用目录启动json服务器：  
```sh
npx json-server --port 3001 --watch db.json
```

将json-server安装为开发依赖项  
```sh
npm install json-server --save-dev
```
安装为开发依赖项之后启动json服务器的命令：  
```
npm run server
```
