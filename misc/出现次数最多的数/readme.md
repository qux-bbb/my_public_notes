# 出现次数最多的数

## 题目
```
问题描述
　　给定n个正整数，找出它们中出现次数最多的数。如果这样的数有多个，请输出其中最小的一个。
输入格式
　　输入的第一行只有一个正整数n(1 ≤ n ≤ 1000)，表示数字的个数。
　　输入的第二行有n个整数s1, s2, …, sn (1 ≤ si ≤ 10000, 1 ≤ i ≤ n)。相邻的数用空格分隔。
输出格式
　　输出这n个次数中出现次数最多的数。如果这样的数有多个，输出其中最小的一个。
样例输入
6
10 1 10 20 30 20
样例输出
10
```

## 思路

```
/*出现次数最多的数

一开始没有思路，想偏了，而且看了一下答案，根本不知道怎么回事
现在又重新理一下思路
要求：输出这n个次数中出现次数最多的数。如果这样的数有多个，输出其中最小的一个
要输出次数最多的数中最小的一个，那就先排大小吧，从小到大排好了
之后找出现次数最多的，这样的话就是要遍历一下，遍历过程中比较，记录
设置了一个number记录当前最符合条件的数，total用来记录当前最多次数 */
```

## 代码

```cpp
#include<iostream>  
using namespace std;  
  
int main(){  
    int n;  
    cin>>n;  
    int *a=new int [n];   //创建数组   
  
          
    for(int i=0;i<n;i++)   //读入数据   
        cin>>a[i];  
      
    int temp;  
    for(int i=0;i<n-1;i++)     //从小到大排序，没有优化   
        for(int j=i+1;j<n;j++)  
            if(a[i]>a[j]){  
                temp=a[i];  
                a[i]=a[j];  
                a[j]=temp;  
            }  
              
    int number=a[0],sum=1,total=0,i=0;  
    while(i<n-1){  
        while(a[i]==a[i+1]){    //找一个数出现的次数，因为已经排序，所以比较好找   
            sum++;  
            i++;  
        }  
              
        if(total<sum){         //比较此数出现次数跟之前最多次数，符合条件则更新   
            total=sum;  
            number=a[i];  
        }  
        if(total==sum&&a[i]<number){  //判断是否次数跟之前最大次数相等且此数较小 ，符合条件则更新   
            number=a[i];  
        }  
        sum=1;    //重置计数变量   
        i++;       //i+1以使进行下一个数的统计   
    }  
    cout<<number;   //输出结果   
    return 0;  
}  
```
