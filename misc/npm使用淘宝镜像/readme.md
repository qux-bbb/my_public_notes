# npm使用淘宝镜像

keywords: npm 源  

npm安装模块的时候，使用默认镜像比较慢，可以使用淘宝镜像。  

```r
# 项目安装所有模块
npm install --registry=http://registry.npmmirror.com
# 安装单个模块示例
npm install express --registry=http://registry.npmmirror.com
```
