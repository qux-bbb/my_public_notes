# photoshop---图片分割制作gif图

图片放到ps里，  
选择 `切片选择工具`，  
`菜单栏->选择->划分` 即可划分为若干张图片，  
`文件->存储为Web所用格式`，放到一个文件夹里  

`文件->脚本->将文件载入堆栈`，全选即可导入  
`窗口->时间轴`  
`创建帧动画` 时间轴右上角箭头打开 `从图层建立帧`  
这时候的动画是反向的，继续 时间轴右上角箭头打开 `反向帧`  

`文件->存储为Web所用格式`，选择gif存储即可
