# IDA---func相关操作

可以自己转data为func，  
Edit -> Functions -> Create function  
有时候没用的话，直接在函数开头按快捷键 P  

可以设置函数结尾，  
Edit -> Functions -> Set function end  

可以设置没有return，  
Edit -> Functions -> Edit function -> Does not return  

暂时只觉得这几个有用  

还有一个好像可以改sp的值，还不会用  
Edit -> Functions -> Change stack pointer
