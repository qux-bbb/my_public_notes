keywords: ollama  

Cherry Studio 是一款支持多个大语言模型（LLM）服务商的桌面客户端，兼容 Windows、Mac 和 Linux 系统。  

官网: https://cherry-ai.com/  
github地址: https://github.com/CherryHQ/cherry-studio
