# IDA---一些快捷键

```
Ctrl + S        Jump to segment, 跳到节
Shift + F12     查找字符串
Alt + T         全文搜索
Ctrl + T        搜索下一个

A               将数据识别为字符串
C               将数据识别为代码
H               16进制和10进制互相转换
R               16进制转化为字符串

X               交叉引用 

空格            切换视图(汇编代码 逻辑图)
Tab             切换视图(汇编代码 伪代码)

F2              下断点
F9              开始调试
Ctrl + F2       重新调试

G               根据地址跳转

Alt + L         开始选择，然后鼠标点到哪里就选到哪里，很方便

Shift + E       导出数据
```
