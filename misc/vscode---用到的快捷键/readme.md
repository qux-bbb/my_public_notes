# vscode---用到的快捷键

## `多行编辑`
Ctrl + Alt + 方向键  
Shift + Alt + 鼠标  

## `行操作`
向上复制行：Shift + Alt + 上箭头  
向下复制行：Shift + Alt + 下箭头  
上移当前行：Alt + 上箭头  
下移当前行：Alt + 下箭头  
删除当前行：Ctrl + Shift + K  

## `格式化`
Shift + Alt + F  

## `前进/后退`
也就是光标的上一个位置/下一个位置  
前进: Alt + LeftArrow  
后退: Alt + RightArrow  

## `匹配括号跳转`
Ctrl + Shift + \  

## `按文件名查找`
Ctrl + P  

## `保持打开状态`
Ctrl + K + Enter  

## `选择颜色主题`
Ctrl + K + T  

## `缩放界面`
Ctrl + "+/-"  

## `只调整编辑文字大小`
设置中勾选"Mouse Wheel Zoom"，然后可以 Ctrl + 鼠标滚轮 调整  

## `显示命令托盘`
Ctrl + Shift + P  

## `折叠(fold)`
折叠: Ctrl + Shift + [  
展开: Ctrl + Shift + ]  
切换折叠: Ctrl + K + L  
全部展开: Ctrl + K + J  
全部折叠: Ctrl + K 和 Ctrl + 0  

## `变量重命名`
F2
