# 通用搜索技巧

提取关键字，尽量独特  

## 搜索引擎
百度：国内相关的事、简单问题解决  
谷歌：比较通用  

都可以限制时间范围  

## 一些站点介绍
百度一般有用的信息在：百度知道、cnbogs、CSDN  
- 百度知道：基础的通俗解释  
- cnblogs：一般有比较详细可靠的解决方法  
- CSDN：需要谨慎识别、站点广告太多  

谷歌一般有用的信息在：维基百科、stackoverflow、个人博客、github  
- 维基百科：基础的详细解释  
- stackoverflow：程序员搜索解决问题的好去处  
- 个人博客：可能有详细可靠的解决方法  
- github：很多实用的项目
