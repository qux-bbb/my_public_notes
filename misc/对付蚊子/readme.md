# 对付蚊子

最有用的就是蚊帐，没有蚊帐退而求其次就是花露水。  

六神驱蚊花露水  
六神止痒花露水  

- 一开始知道蚊子很多，直接全身喷驱蚊花露水，然后睡觉  
- 半夜被蚊子咬醒，被咬的地方喷止痒花露水，然后全身喷驱蚊花露水，睡觉  

被咬的地方千万不要抓，越抓越痒，抓破会花挺长时间恢复  

徒手拍蚊子可以上下夹击，命中率很高
