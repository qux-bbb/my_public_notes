# pycharm---code-Templates

设置一下新建python文件的默认内容  

File -> Settings -> Editor -> File and Code Templates -> Python Script  

我的模板:  
```python
#!usr/bin/env python
# coding:utf8

"""
@author: hello
@file: ${NAME}.py
@time: ${YEAR}/${MONTH}/${DAY}
"""


def main():
    print('Hello World!')


if __name__ == '__main__':
    main()
```
