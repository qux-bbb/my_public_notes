# 计算机系统结构一些缩写

```
RISC    Reduced Instruction Set Computer    精简指令集计算机
CISC    Complex Instruction Set Computer    复杂指令集计算机


SISD    Single Instruction stream Single Data stream    单指令流单数据流
SIMD    Single Instruction stream Multiple ultimateData stream    单指令流多数据流
MISD    Single Instruction stream Single Data stream    多指令流单数据流
MIMD    Multiple Instruction stream Multiple Data stream    多指令流多数据流

CPI    Cycles Per Instruction    每条指令的平均时钟周期数
IPC    Instruction Per Cycle    每个时钟周期完成的指令条数

IC     Instruction count    所执行指令条数

DMA    Direct Memory Access    直接存储器访问

ALU    Arithmetic Logical Unit    运算器

TP    ThroughPut    吞吐率

RAW    Read After Write    写后读冲突
WAW    Write After Write    写后写冲突
WAR    Write After Read    读后写重突

ILP    Instruction-Level Parallelism    指令级并行

BHT    Branch History Table    分支历史表

BTB    Branch Target Buffer    分支目标缓冲器

ROB    ReOrder Buffer    前瞻执行缓冲器

LRU    Least Recently Used    最近最少使用法


MTTF    Mean Time To Failure    平均失效前时间
MTTR    Mean Time To Repair    平均修复时间
MTBF    Mean Time Between Failures    平均失效间隔时间 

CDB    Common Data Bus    公共数据总线

```
