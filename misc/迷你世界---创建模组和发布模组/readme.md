# 迷你世界---创建模组和发布模组

实测1.53.1版本可用

入口：  
开始游戏 -> 创建新世界 -> 创作工具 -> 创作工具3.0 -> 开始创作


## 创建模组
编辑触发器：
1. 点击左上角扳手图标，点击“触发器”，拖动修改逻辑
2. 修改完成后，点击左下角播放按钮可测试效果，点击左下角暂停按钮可退出测试
3. 点击左侧第四个长得像记事本的图标，可以修改组件名称，这里假设改为hello
4. 点击左上角退出图标，可以退出触发器编辑界面

创建模组：
1. 点击左上角扳手图标，点击“模组”，进入模组管理界面
2. 点击“创建模组”，输入模组名称和描述，点击“下一步”
3. 点击右上角“新增”，确认“hello”组件被勾选，关闭后点击“下一步”，直到完成即可


## 发布模组
1. 提前准备1张封面图和至少3张详情图
2. 点击左侧“我创建的”，可以查看自己创建的模组
3. 选择未发布的模组，点击“发布”，输入封面图和详情图，添加标签，勾选同意协议，点击“发布”即可

等审核通过后即可发布
