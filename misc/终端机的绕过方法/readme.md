# 终端机的绕过方法

```
0x00 记录下看的东西

0x01  
mailto: 
就找各种能发邮件的地方，然后就可以跳出沙箱

右键绕过
右键可以通过长按，两个手指，三个手指等打开，然后 打印 调 打印什么的，可以看到整个电脑的目录，有的还可以跳出沙箱

操作错误或输入错误信息
 使跳出错误提示，或者利用输入法，或者利用任务栏，跳出沙箱

超链接
访问超链接可以直接用浏览器打开网页，然后关掉网页可以进入电脑桌面环境

0x02
总结：基本就是右键，链接，错误，想办法跳出来就可以了

0x03
详细的来源网址：http://drops.wooyun.org/papers/346
```
