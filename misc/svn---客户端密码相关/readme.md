# svn---客户端密码相关

用户名密码保存在这个文件里:  
`~/.subversion/auth/svn.simple`  

全都是明文, 所以小心  

清除保存的用户名密码, 只要删除上面说的那个文件就好了  
为了简单, 直接把`~/.subversion/auth`文件夹删了也可以  

只能由服务器分配用户名密码
