# SM系列算法了解

SM, 商密，商用密码，中国发展的一系列密码算法。  

SM1, 硬件芯片使用的对称密码，不公开。  
SM2, 椭圆曲线公钥密码算法，非对称。  
SM3, 密码杂凑算法。  
SM4, 分组密码算法，对称。  
SM9, 标识密码算法，非对称。  
ZUC, 祖冲之流密码算法，对称。
