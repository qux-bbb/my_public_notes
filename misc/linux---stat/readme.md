# linux---stat

keywords: 文件属性 文件详细属性  

stat, status, 获取文件/文件夹的信息，如文件大小/创建时间/修改时间等。  

示例：  
```r
> stat hello.txt
  文件：hello.txt
  大小：6               块：8          IO 块：4096   普通文件
设备：812h/2066d        Inode：9317039     硬链接：1
权限：(0644/-rw-r--r--)  Uid：( 1000/   alice)   Gid：( 1000/   alice)
最近访问：2021-09-10 21:02:30.540472525 +0800
最近更改：2021-09-10 21:02:30.540472525 +0800
最近改动：2021-09-10 21:02:30.540472525 +0800
创建时间：2021-09-10 21:02:30.540472525 +0800
> stat hello_lnk
  文件：hello_lnk -> /home/alice/files/tmp/hello.txt
  大小：31              块：0          IO 块：4096   符号链接
设备：812h/2066d        Inode：9317040     硬链接：1
权限：(0777/lrwxrwxrwx)  Uid：( 1000/   alice)   Gid：( 1000/   alice)
最近访问：2021-09-10 21:05:01.239370926 +0800
最近更改：2021-09-10 21:05:01.239370926 +0800
最近改动：2021-09-10 21:05:01.239370926 +0800
创建时间：2021-09-10 21:05:01.239370926 +0800
```
