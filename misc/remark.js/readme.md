# remark.js

```
一个可以用markdown语法，通过浏览器展示的PPT，使用的时候只要引入remark.js即可
下载地址：https://github.com/gnab/remark
使用方法：下载之后看到 boilerplate-local.html等3个页面，
local：引用本地的remark.js
remote: 引用远程的remark，需要联网才能有效果
single：把js直接写在页面里

也就是说，有一个html页面和一个js文件（single，remote不需要）就可以了，其他文件就可以删掉了

好好玩吧


谢谢ls推荐的小工具
```
