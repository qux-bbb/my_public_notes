# CentOS

Community Enterprise Operating System，社区企业操作系统  
Red Hat Enterprise Linux依照开放源代码规定释出的源代码所编译而成  

官网: https://www.centos.org/  

查看版本  
```r
cat /etc/centos-release
```

包管理工具：yum  
Yellow dog Updater, Modified
