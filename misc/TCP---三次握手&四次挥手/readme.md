# TCP---三次握手&四次挥手

## 三次握手
```
|-----|    SYN     |-----|
|     |----------->|     |
|     |   SYN+ACK  |     |
|  A  |<-----------|  B  |
|     |    ACK     |     |
|     |----------->|     |
|-----|            |-----|
```

## 四次挥手

```
|-----|   FIN   |-----|
|     |-------->|     |
|     |   ACK   |     |
|     |<--------|     |
|  A  |   FIN   |  B  |
|     |-------->|     |
|     |   ACK   |     |
|     |<--------|     |
|-----|         |-----|
```
