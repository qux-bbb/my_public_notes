# AndroidStudio一些东西

```
如果两个模拟器用了同一个镜像，不能同时开
--------------------------------------------------------------------------------
Android Studio 快捷键
Crlt+D：复制一行
Ctrl+X：删除一行
Alt+Enter：代码修正
Crtl+Alt+L：格式化代码 （不知道为什么不起作用了  
选中这行代码→Code→Reformed Code 手动格）
Shift+Enter：另起一行
Alt+鼠标左键：列编辑
Ctrl + Shift + Enter：补齐一行最后的分号并跳到行尾
--------------------------------------------------------------------------------
打开DDMS
Tools-->Android-->Android Device Monitor
在DDMS下的 File Explorer选项下可以在电脑和模拟器之间传递文件
--------------------------------------------------------------------------------
更改字大小
File --> Setting --> Editor --> colors&Fonts --> Font
然后另存一个主题，就可以自己修改了
--------------------------------------------------------------------------------
如果变成全角字符怎么办
切换几次输入法就好了
--------------------------------------------------------------------------------
配一下环境变量吧
新建：
android
D:\Android\sdk\platform-tools;D:\Android\sdk\tools
path中添加：
%android%
--------------------------------------------------------------------------------
更改 minSdkVersion(兼容的Android最低版本)
在 Gradle Scripts 下的 build.gradle(Module:app) 中
```
