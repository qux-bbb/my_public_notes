# blender---图片序列和音频合成视频

```
文件 -> 新建 -> Video Editing

下方的 序列编辑器,
添加 -> 图像/序列
添加 -> 声音

要注意添加图像序列时设置好帧范围, 文件排序从第1帧到最后一帧

右侧设置输出属性(图标像一张图片从盒子伸出来)
规格尺寸 设置起始帧/结束帧/帧率, 帧率先设置30即可
设置文件格式为 FFmpeg video
编码 -> 容器 选择 MPEG-4
音频 -> 音频编码器 选择 MP3

参考链接:
https://www.youtube.com/watch?v=LmxaYwmewWs
https://www.blender.org/forum/viewtopic.php?t=19452
```
