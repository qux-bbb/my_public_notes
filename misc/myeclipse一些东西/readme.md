# myeclipse一些东西

```
--------------------------------------------------------------------------------
更改默认workspace

File--switch workspace --other
即可更改
--------------------------------------------------------------------------------
编码

windows→preference→workspace（最大的那个)

然后找到Text  file  encoding  改为UTF-8
--------------------------------------------------------------------------------
修改项目访问路径
打开Mye，找到你要修改的项目，左键点击后，点击右键，弹出属性，点击properties
在搜索框搜索web，并双击web
看到web context root，在这里修改访问路径，修改完点击ok即可
参考：http://jingyan.baidu.com/article/eae07827a945b61fed548544.html
--------------------------------------------------------------------------------
快捷键
Ctrl + Shift + ↓   复制一行
Ctrl + D              删除一行
--------------------------------------------------------------------------------
提示功能（补全功能）
Alt + / 
输到一半的时候（或者只输几个字符）  Alt + /   可以列出符合已经输的，然后选择就可以了
注意的是不能把名字全输完再按，那样没有提示......这种情况一般出现在你输完名字但是想要补全后面的，但是找不到了，所以别输全...
```
