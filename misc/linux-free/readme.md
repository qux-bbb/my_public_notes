free, 查看可用内存大小，可显示总内存、已使用内存、剩余内存，信息是从/proc/meminfo获取整理的  

一次执行如下：  
```
hello@hello-PC:~/Desktop$ free -h
              total        used        free      shared  buff/cache   available
Mem:          3.8Gi       884Mi       1.4Gi        86Mi       1.6Gi       2.6Gi
Swap:         6.0Gi          0B       6.0Gi
```
