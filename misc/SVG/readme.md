# SVG

SVG, Scalable Vector Graphics, 可缩放矢量图形，基于 XML 标记语言，用于描述二维的矢量图形。  

示例：
```svg
<svg>
    <circle cx="100" cy="50" r="40" stroke="black" stroke-width="2" fill="red" />
</svg>
```
