# ftp---命令行简单命令

连接ftp服务器   
```sh
ftp 1.2.3.4
# 按提示输入用户名密码即可
```


列目录  
```sh
# 远程目录
dir  # 或者 ls

# 本地目录
!dir
```


切目录  
```sh
# 切远程目录
cd

# 切本地目录
lcd
```


设置文件传输模式  
```sh
# 有2种模式
ascii  # 自动转换内容以适配系统
binary  # 原始传输
```


下载文件到本地  
```sh
get a.py
```


上传文件到服务器  
```sh
put a.py b.py  # 或者send
```
