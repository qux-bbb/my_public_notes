# 从源代码编译python

下载地址：  
https://www.python.org/downloads/  

安装方法：  
```bash
tar xzvf Python-2.7.11.tgz
cd Python-2.7.11
./configure --prefix=/usr --enable-shared --with-system-ffi --enable-unicode=ucs4
make
sudo make install
```
