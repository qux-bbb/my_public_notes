# android---一些东西

```
--------------------------------------------------------------------------------
四大组件
Activity
Service（跟Activity相比，没有界面）
Broadcast Receiver
Content Provider

--------------------------------------------------------------------------------
Intent
可以在Activity启动Service或者别的Activity时，传递信息

--------------------------------------------------------------------------------
Thread
创建新的线程，做一些耗时操作

--------------------------------------------------------------------------------
handler
在多线程时，可以通过handler传递信息到主线程，更改主Activity界面
（也有封装好的AsyncTask）

--------------------------------------------------------------------------------
Toast
一些提示信息，不能获取焦点
```
