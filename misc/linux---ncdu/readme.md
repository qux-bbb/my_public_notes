# linux---ncdu

keywords: 文件夹大小  

ncdu, NCurses Disk Usage, 一个可交互的文件夹占用空间统计工具，NCurses版本的du。  

官网: https://dev.yorhel.nl/ncdu  
github地址: https://github.com/rofl0r/ncdu  

ubuntu安装：  
```r
sudo apt install ncdu
```

左右箭头可以切换上下级目录，q键退出。
