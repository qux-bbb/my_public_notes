# codeblocks---一些东西

```
应该下载带mingw的，
官网：http://www.codeblocks.org/


恢复初始界面
View→Perspectives→default
不行的话 选择  delete current 也可以
--------------------------------------------------------------------------------
重命名
法1：选中某个词，Ctrl + R
法2：选中某个词，右键→Code Refactoring→Rename symbols
--------------------------------------------------------------------------------
注释
Ctrl + Shift + C  注释多行
Ctrl + Shift + X  取消注释
--------------------------------------------------------------------------------
调试 查看变量值
Debug→Debugging windows→Watches
--------------------------------------------------------------------------------
关掉拼写检查
Settings→Editor...→左侧的 SpellChecker
然后把最上面的三个复选框点掉，保存即可
--------------------------------------------------------------------------------
复制删除一行
Ctrl + D 复制当前行到下一行
Ctrl + L  删除一行
--------------------------------------------------------------------------------
格式化代码
右键，Format use AStyle
--------------------------------------------------------------------------------
编译运行
F9
```
