# python---各进制间的转换

```r
# python 2.6以后内置函数
#10进制转为2进制
>>> bin(10)
'0b1010'

#2进制转为10进制
>>> int("1001",2)
9

#10进制转为16进制
>>> hex(10)
'0xa'

#16进制到10进制
>>> int('ff', 16)
255
>>> int('0xab', 16)
171

#十进制转为八进制
>>print("%o"%10)
>>12

#16进制到2进制
>>> bin(0xa)
'0b1010'

#10进制到8进制
>>> oct(8)
'010'

#2进制到16进制
>>> hex(0b1001)
'0x9'
```
