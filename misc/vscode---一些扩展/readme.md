# vscode---一些扩展

记一些比较好用的扩展  

## `Markdown All in One`
方便写markdown  
可生成目录(TOC)  
有TODO快捷功能, `Alt + C`标记  

## `Markdown PDF`
转换markdown到pdf, html, png 或者 jpeg  

## `Path Intellisense`
自动完成路径  

## `Asciiflow2`
字符流程图

## `show-offset`
右下角显示当前光标在文件中的偏移  

## `Project Manager`
相当于一些常用文件夹的快捷方式，打开很方便  

## `Text Utils`
自己开发的插件，描述是 "Simple Text Utils"，一些常用的文本处理功能
