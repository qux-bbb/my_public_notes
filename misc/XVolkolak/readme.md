# XVolkolak

XVolkolak可用于检测和脱壳。  

发布站点: https://horsicq.github.io/#xvolkolak  
下载压缩包名称示例: xvlk_win32_public_0.22.zip  

支持以下壳和其他可能的壳：  
```r
AHPacker
ASPack
BeRoEXEPacker
ExeFog
!EP(EXE Pack)
Fish PE Packer
FSG
kkrunchy
MEW
MPRESS
nPack
NSPack
Packman
PECompact
Petite
QuickPack NT
PeX
UPX
(Win)Upack
```
