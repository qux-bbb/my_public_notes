# vim简单使用

VIM - Vi IMproved, 一个编辑器  
vi - visual interface, 一个编辑器  

可以执行命令 `vimtutor` 练习基本的操作，下面记一些之前用到的  

```
设置行号
:set nu     # 设置显示行号
:set nonu   # 取消显示行号

跳转到某行
:108    # 跳转到第108行
108G    # 跳转到第108行方法2

撤销&反撤销
u           # 撤销
Ctrl + R    # 反撤销

跳转
gg  # 到顶部 
G   # 到底部

复制
yy  # 1行
3yy # 3行

粘贴
p

删除字符
x

删除单词
dw  # 当前光标后的单词部分
daw # 光标所在整个单词

删除行
dd  # 1行
3dd # 3行

选中
v 释放之后  按方向

选中多行
V 释放之后 按方向

选中块区域
Ctrl + v  释放之后 按方向

基本移动--左下上右
hjkl

按word向后(前)移动
w(b)
按word向后(前)移动-忽略符号
W(B)
e也是后移，会移到word的最后一位

显示模式到插入模式
i	# 当前光标处插入
a	# 当前光标后一个字母处插入
I	# 行首插入
A	# 行尾插入

查找文本
:/hello	# 从前往后搜索hello
:?hello	# 从后往前搜索hello
按n到下一个，按N到上一个

替换字符
r  # 单个字符
R  # 连续替换字符

高级替换
:s/foo/bar/g    # 当前行
:3,5s/foo/bar/g # 3-5行
:%s/foo/bar/g   # 全文

询问用户是否替换
:%s/foo/bar/gc  # 全文

统计全文指定内容个数
:%s/yes//gn

取消折行
:set nowrap  # 取消自动折行后, 复制可能会丢失内容


查看配对括号
%

执行外部命令
:!ls

命令提示补全
<TAB>       # 切换可能的命令
Ctrl + D    # 显示所有可能的命令

显示帮助
:help       # 帮助文档
:help w     # 指定命令的帮助
```
